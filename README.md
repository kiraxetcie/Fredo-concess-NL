# Fredo-concess-NL
Permet de vendre un véhicule de manière simple et efficace

# Installation

Très simple, rendez vous ici: https://github.com/kiraxetcie/Fredo-concess-NL/blob/main/concessplugin.dll
Puis glissez votre fichier vers votre fichier Plugins de votre serveur

# Fonctionnement

Lancez une fois votre plugin puis modifiez le fredoconcessjs.json créé avec l'id de votre entreprise
Une personne qui sera dans l'entreprise choisie pourra faire /concess et ainsi vendre un véhicule via l'id de celui ci.

# Droits

Utilisation sur tout serveurs autorisé sauf avec un but commercial.
